# 仓库管理系统简介 python864

本项目是一款基于 Python + Flask + MySQL 技术栈的仓库管理系统。

## 技术栈

- **后端语言与框架**：Python 3，Flask
- **数据库系统**：MySQL
- **开发工具**：PyCharm
- **数据库连接工具**：Navicat

## 功能模块

- **系统管理**：
  - 管理员登录
  - 修改密码
- **仓库操作**：
  - 货物增加
  - 货物删除
  - 货物入库
  - 货物销售
- **库存盘点**：
  - 盘点功能

## 系统角色

- 管理员：负责系统的管理与仓库的日常操作

## 运行环境

- Python 3 环境配置
- Flask 框架安装
- MySQL 数据库环境
- PyCharm 开发工具

## 部署步骤

1. 使用 PyCharm 导入项目
2. 安装项目所需依赖包
3. 配置好 MySQL 数据库
4. 正常运行项目
